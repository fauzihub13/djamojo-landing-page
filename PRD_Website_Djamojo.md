## 1. Ringkasan Eksekutif (Executive Summary)

Djamojo by Djamoe Spirit adalah UMKM pionir produsen jamu tradisional homemade siap minum di Kota Bandung yang berdiri sejak 2012. Sejak awal, Djamojo telah mengadopsi kemasan botol kaca sebagai wujud komitmen menjaga autentisitas rasa, higienitas, dan keberlanjutan lingkungan. Website ini berfungsi sebagai **etalase digital utama** yang menyeimbangkan dua kekuatan:

1. **Branding Emosional** — menceritakan kisah otentik, nilai heritage, dan karakter unik Djamojo untuk membangun ikatan emosional dengan konsumen.
2. **Konten Informatif-Akademik** — mendokumentasikan secara transparan program sirkular (return bottle, sterilisasi food-grade, strategi komunikasi hijau) berbasis riset dan regulasi, untuk membangun kredibilitas serta trust.

**Nilai unik (USP):** Satu-satunya brand jamu Bandung yang menggabungkan *autentisitas resep tradisional* dengan *sistem ekonomi sirkular* berstandar food-grade, terdokumentasi secara akademis.

## 4. Tujuan Website (Website Goals)

### 4.1. Tujuan Bisnis
1. **Meningkatkan penjualan online** (WhatsApp order, marketplace, atau pre-order).
2. **Memperkuat brand recall & brand trust** sebagai pionir jamu sirkular di Bandung.
3. **Mengonversi pengunjung menjadi anggota program Return Bottle** (akuisisi Green Points member).
4. **Memperluas jangkauan pasar** ke luar Bandung dengan edukasi program return yang transparan.
5. **Menarik mitra ritel, akademisi, dan media** melalui konten berbasis riset.

### 4.2. Tujuan Konten / Akademik
1. Mendokumentasikan SOP Return Bottle sebagai **etalase transparansi**.
2. Menyajikan **landasan ilmiah** (reduksi emisi 90%, Permen LHK P.75/2019, SDGs 12.5).
3. Menjadi **ruang edukasi** tentang ekonomi sirkular di sektor UMKM minuman.

## 7. Strategi Konten (Content Strategy)

### 7.1. Prinsip Konten: **"Edukatif tanpa menggurui, branding tanpa berlebihan"**

Setiap halaman mengikuti formula **4E**:
- **Empati** — pahami keraguan konsumen (harga, higienitas, rasa).
- **Edukasi** — berikan data, proses, dan referensi.
- **Emosi** — ceritakan kisah, dampak, dan nilai.
- **Endorse** — dorong aksi (pesan, kembalikan botol, share).

### 7.2. Pemetaan Konten PDF → Halaman Website

| Sumber dari Booklet PDF | Halaman Website | Format |
|---|---|---|
| Pendahuluan / Krisis Sampah Plastik (Bagian I) | Edukasi & Riset → "Krisis Sampah Plastik" | Artikel + infografis data SIPSN KLHK |
| Landasan Regulasi & SDGs 12.5 (Bagian I.2) | Edukasi & Riset → "Regulasi & SDGs" | Timeline interaktif |
| Komitmen Hijau Djamojo (Bagian II) | Brand → Cerita Djamojo | Storytelling |
| Fakta Ilmiah Botol Kaca (Gheewala, 2023) | Program Return Bottle → "Mengapa?" | Infografis (reduksi emisi 90%) |
| 4 Tujuan Strategis Return Bottle (Bagian II.1–4) | Program Return Bottle → Hero section | 4 ikon + caption |
| SOP Distribusi (Bagian III) | Program Return Bottle → "SOP Distribusi" | Step-by-step + foto |
| SOP Alur Pengembalian (Bagian IV) | Program Return Bottle → "Alur Pengembalian" | Diagram alur (flowchart) |
| Skema Insentif (Bagian IV.3) | Program Return Bottle → "Green Reward" | Tabel + kalkulator poin |
| SOP Sanitasi (Bagian V) | Program Return Bottle → "Sanitasi Food-Grade" | Visualisasi proses 4 tahap |
| Strategi Komunikasi & Hashtag (Bagian VI) | Blog + Press Release | Highlight campaign #KembaliBerseri |

### 7.3. Jenis Konten
- **Artikel panjang (long-form):** 1.500–2.500 kata, terstruktur H2/H3, SEO-friendly.
- **Infografis:** 6–10 visual utama (alur return, siklus botol, proses sterilisasi, perbandingan emisi).
- **Video pendek (Reels/Shorts):** behind-the-scenes produksi, sterilisasi, testimoni.
- **Studi kasus & white paper:** untuk audiens akademisi & mitra.
- **User-generated content:** galeri pelanggan + botol kembali.

## 8. Desain & Pengalaman Pengguna (Design & UX)

### 8.1. Prinsip Desain
- **Warm Minimalism:** putih bersih, sentuhan earth-tone, banyak whitespace.
- **Visual Storytelling:** foto otentik produksi, bukan stok generik.
- **Mobile-First:** 70%+ trafik diprediksi dari ponsel.
- **Aksesibilitas:** kontras WCAG AA, alt-text, struktur heading semantik.

### 8.2. Komponen Kunci
- **Hero Homepage:** foto produk botol kaca di atas kayu/kain, headline brand + CTA "Pesan Sekarang" & "Pelajari Return Bottle".
- **Sticky CTA WhatsApp** di pojok kanan bawah (mobile).
- **Progress Tracker Green Points** (visual, fase 2).
- **Knowledge Base Interaktif** untuk halaman edukasi (TOC sticky, glossary, sitasi hover).

### 8.3. Microcopy Guidelines
- Hindari jargon ilmiah tanpa penjelasan (mis. "autoclave 121°C" → "sterilisasi uap suhu tinggi food-grade").
- Setiap klaim keberlanjutan harus ada data/sumbernya.
- Gunakan bahasa inklusif & hangat ("kami" bukan "perusahaan kami").

## 9. Spesifikasi Teknis

### 9.1. Stack (Rekomendasi)
- **Frontend:** Next.js 14 (App Router) + TypeScript + Tailwind CSS.
- **CMS:** Sanity.io atau Strapi (untuk artikel blog, edukasi, dan pembaruan tanpa redeploy).
- **Animasi:** Framer Motion (transisi halus pada scroll & hover).
- **Gambar:** Next/Image + WebP/AVIF, lazy load.
- **Form:** React Hook Form + Zod (validasi kontak, pendaftaran member).
- **Hosting:** Vercel (otomatis CDN, edge functions).
- **Analytics:** Plausible.io (privacy-friendly, tanpa cookie banner yang mengganggu).
- **SEO:** Next SEO + sitemap.xml + JSON-LD structured data (Organization, Article, FAQ).

### 9.2. Performa
- Lighthouse Score target: **≥ 95** di semua metrik (Performance, Accessibility, Best Practices, SEO).
- LCP < 2.5s, CLS < 0.1, INP < 200ms.

### 9.3. Keamanan & Privasi
- HTTPS wajib (Let's Encrypt).
- Tidak ada pelacakan pihak ketiga tanpa consent.
- Kebijakan Privasi & Syarat Pemesanan sesuai UU PDP.

### 9.4. Multibahasa
- Struktur URL: `/id/` dan `/en/` (default redirect ke `/id/`).
- Konten PDF dalam Bahasa Indonesia; versi English untuk audiens akademisi internasional.

## 14. Risiko & Mitigasi

| Risiko | Dampak | Mitigasi |
|---|---|---|
| Konten PDF terlalu akademis, konsumen awam bingung | Tinggi | Sediakan dua versi: "Ringkas" (konsumen) & "Lengkap" (akademisi) pada halaman yang sama |
| Keterbatasan foto/video produksi | Sedang | Kerjakan sesi foto profesional paralel dengan dev |
| Kepercayaan konsumen terhadap higienitas botol reuse | Tinggi | Tampilkan transparansi proses sterilisasi (foto, video, sertifikat) + sertifikasi BPOM jika memungkinkan |
| Tim kecil, scope melebar | Sedang | Tunda fitur P2 jika fase 1 belum stabil |
| Kompetitor menyalin konsep | Rendah | Fokus pada eksekusi & komunitas; USP sulit ditiru |
| Perubahan regulasi | Rendah | Pantau Permen LHK; subsite "Regulasi" jadi rujukan update |

- **Sustainability Signaling** — isyarat keberlanjutan dalam komunikasi merek.
- **3R (Reduce, Reuse, Recycle)** — prinsip ekonomi sirkular.
